# Stenvik Lead Pipeline

Автоматический сбор B2B-лидов для цифровой студии Stenvik.
Источник: HH.ru работодатели. AI-анализ: Claude Sonnet 4.6. Хранилище: SQLite.
Интерфейс: FastAPI + Bootstrap.

## Что делает

1. Проходит по работодателям HH.ru в выбранных городах.
2. Исключает IT-компании (они не целевые).
3. Для каждой компании — ищет сайт, качает главную, определяет стек (старый CMS / Tilda / нет HTTPS / нет сайта вообще).
4. Отдаёт данные Claude Sonnet — тот анализирует компанию как потенциального клиента Stenvik:
   - Краткая сводка
   - 3-5 конкретных болей
   - Рекомендуемые услуги из прайс-листа Stenvik
   - **Персональный хук для первого контакта продажника**
   - Приоритет 1-5
5. Сохраняет в SQLite, показывает в веб-интерфейсе.

## Стоимость
- HH.ru API: бесплатно
- Claude Sonnet 4.6 с prompt caching: ~2₽ на лид
- При 30-50 лидов/день: ~60-100₽/день, ~2-3k₽/мес

## Запуск локально

```powershell
# 1. Создать виртуальное окружение и установить зависимости
py -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt

# 2. Скопировать .env.example → .env и вписать ANTHROPIC_API_KEY
copy .env.example .env
# отредактировать .env

# 3. Запустить веб-интерфейс
python run.py
# Открыть http://127.0.0.1:8000, войти под admin:admin123 (из .env)

# 4. Вручную запустить пайплайн (5 лидов для теста)
python run.py pipeline 5
```

Планировщик (каждые 15 мин) стартует автоматически при запуске веб-приложения.

## Структура

```
lead_pipeline/
├── app/                 # FastAPI (web UI, auth, config, модели БД)
├── scrapers/            # HH.ru и веб-скрейпер
├── ai/                  # Claude Sonnet: промпт + анализатор
├── pipeline/            # Оркестратор и APScheduler
├── data/                # SQLite база (leads.db)
├── deploy/              # nginx / systemd конфиги для VPS
├── run.py               # Точка входа
├── requirements.txt
└── .env.example
```

## Деплой на VPS

См. `deploy/` — nginx + gunicorn + systemd.

## Как пользоваться продажникам

1. Открыть главную — там топ-10 горячих лидов.
2. Нажать на лид → увидеть "боли", "рекомендованные услуги", "хук для первого контакта".
3. В правой колонке — сменить статус (в работе / связались / сделка / отказ), записать заметку.
4. Фильтры на `/leads` — по приоритету, статусу, городу.
