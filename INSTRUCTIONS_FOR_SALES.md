# Инструкция продажнику — как работать с таблицей лидов

## Где таблица

**Ссылка:** https://yadi.sk/d/HJ5uGucyiJtWuA

Открой → кнопка **"Открыть в Документах"** (сверху) → откроется редактор
Яндекс.Таблиц. Для редактирования нужен свой Яндекс-аккаунт (если нет — сделай
за минуту на yandex.ru).

> Если видишь "только просмотр" — попроси админа дать доступ на редактирование
> (см. раздел ниже).

## Колонки таблицы

| Колонка | Что значит |
|---|---|
| Дата | когда агент нашёл и добавил лид |
| Компания, Город, Индустрия | кто это |
| Сайт | их сайт — **открывай сразу**, чтобы понять контекст |
| Статус сайта | `has_site` / `cms_site` (WordPress/Bitrix) / `constructor_site` (Tilda) / `no_https` / `dead_site` / `no_site` |
| **Приоритет** | 5 = горячий, 1 = не наш. **Работай сверху вниз по приоритету** |
| Причина | почему агент поставил такой балл |
| Описание | 1-2 предложения про компанию |
| Боли | конкретные проблемы с цифровизацией |
| Услуги Stenvik | что предложить (взято из нашего прайса) |
| Хук для продажника | **готовая формулировка для первой реплики** — можно копировать |

## Что заполнять самому (твои колонки)

| Колонка | Как заполнять |
|---|---|
| **Позвонил** | Кликни ячейку → выпадёт список → выбери `✓`. Один клик — и видно что звонок был |
| **Дата звонка** | Пиши когда позвонил: `2026-04-22 14:30` |
| **Ответственный** | Твои инициалы или имя |
| **Статус сделки** | Выпадающий список: Новый / В работе / Связались / Квалифицирован / Сделка / Отказ / Не наш |
| **Фидбэк по звонку** | Свободный текст: что сказал клиент, какие возражения, когда перезвонить, контакт (ЛПР) |

## Цвета строк = приоритет

- 🔴 Светло-красный — приоритет 5 (горячий, звони сегодня)
- 🟡 Светло-жёлтый — приоритет 4 (тёплый, на этой неделе)
- 🟢 Светло-зелёный — приоритет 3 (средний, email-рассылка подходит)
- ⚪ Серый — приоритет 1-2 (не тратить время кроме как для охвата)

## Рабочий день продажника

1. Открыть таблицу утром
2. Отфильтровать столбец "Позвонил" = пусто (галочка "⯆" на шапке → снять "✓")
3. Отсортировать по "Приоритет" от 5 к 1
4. Работать сверху вниз:
   - Прочитать "Хук для продажника"
   - Открыть сайт компании (колонка "Сайт")
   - Позвонить / написать
   - В колонке "Позвонил" поставить `✓`
   - Заполнить "Дата звонка", "Ответственный", "Статус сделки"
   - Если разговор был — в "Фидбэк по звонку" коротко написать что узнал

## Важные правила

1. **Не удаляй строки** — это сломает дедупликацию агента, он может снова
   добавить ту же компанию.
2. **Не трогай первые 12 колонок** (Дата...Хук) — это данные агента, они
   автоматически перезаписываются при его работе только для НОВЫХ строк,
   но если ты поправишь старые — теряется аудит.
3. **Твои колонки (Позвонил...Фидбэк) — твои**. Агент их не трогает никогда.
4. Новые строки добавляются **автоматически** внизу таблицы каждые ~25 минут.

## Если что-то не работает

- Не вижу выпадашек → проверь что открываешь в Яндекс.Документах, а не скачиваешь файл
- Нет доступа на редактирование → пиши админу, он должен добавить твой email
- Поля обрезаны → растяни колонку мышкой за границу
- Таблица не обновляется → обнови страницу (Ctrl+R)

## Как админ раздаёт доступ на редактирование (для справки)

1. Открыть https://disk.yandex.ru
2. Перейти в папку `Stenvik`
3. Правый клик на `leads.xlsx` → "Поделиться"
4. Поставить галочку "Разрешить редактирование"
5. Добавить email продажников → "Отправить"
